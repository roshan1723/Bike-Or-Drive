# Bike-Or-Drive
Generates a map of where its faster to bike rather than drive from a given starting point (in this case the MSC building)
We used google maps API as well as the bokeh library to overlay the area over which it is faster to bike rather than drive.

Download the gmap file and the output will show in browser

This project was made in 24 hours for the TAMUHack 2022 hackathon.
